  • 数组下标都是从0开始的。

总结

使用二分查找的前提：

  • 数组为有序数组

使用二分法的关键是：想清楚区间的定义，区间的定义就是不变量，要在二分查找的过程中，保持不变量，就是在while寻找中每一次边界的处理都要坚持根据区间的定义来操作，这就是循环不变量规则。

二分法第一种写法

定义target在一个左闭右币的区间里，即 [left, right]

区间定义决定二分法的写法，target在区间，所以有以下两点：

  • while(left <= right) 使用 <=，因为 left == right是有意义的
  • if (nums[middle] > target) ,right 要赋值为 middle - 1，因为当前这个nums[middle]一定不是target，那么接下来要查找的左区间结束下标位置就是 middle - 1
  • left同理

二分法第二种写法

target定义在左闭右开的区间里，即[left, right)，则对于二分法的边界处理方式截然不同，有以下两点：

  • while (left < right)，这里使用 < ,因为left == right在区间[left, right)是没有意义的
  • if (nums[middle] > target)，right更新为 middle，因为当前nums[middle]不等于target，去左区间继续寻找，而寻找区间是左闭右开区间，所以right更新为middle，即：下一个查询区间不会去比较nums[middle]。
